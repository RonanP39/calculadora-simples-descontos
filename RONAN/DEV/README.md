# calculo.py

Pequeno utilitário para calcular desconto sobre um preço.

Uso básico (interativo):

```powershell
python "calculo.py"
```

Exemplo com argumentos:

```powershell
python "calculo.py" --preco "1.234,56" --taxa 10
```

Tests:

```powershell
python -m unittest discover -v -s "tests" -p "test_*.py"
```
